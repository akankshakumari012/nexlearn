# 🚀 NexLearn – Gamified Learning Platform

### *Built for Hackathon 2026 🏆 by Syntax Squad*

---

## 📌 Overview

NexLearn is a **gamified learning platform** designed to make education more **interactive, engaging, and fun**.
Instead of traditional boring methods, it uses **quizzes, rewards, and progress tracking** to keep students motivated.

---

## 🎯 Problem Statement

Students often lose interest in traditional learning due to:

* Lack of engagement 😴
* No instant feedback ❌
* No motivation or rewards 💔

---

## 💡 Our Solution

NexLearn solves this by introducing **gamification in learning**:

* 🎮 Interactive quiz system
* ⚡ XP (Experience Points) rewards
* 🔥 Streak system for consistency
* 🧠 Level progression
* ⏱️ Timer-based challenges
* ✅ Instant feedback with correct answers

---

## ✨ Key Features

### 🧩 Learning Experience

* Unlimited quiz questions (randomized)
* Instant answer validation
* Correct answer shown if wrong

### 🎯 Gamification

* XP points system
* Level progression
* Daily streak tracking
* Progress bar visualization

### ⚡ Smart Features

* Timer-based quiz system
* Dynamic greeting (Good Morning / Evening)
* Exit quiz anytime
